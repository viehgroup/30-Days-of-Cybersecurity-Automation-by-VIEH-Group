# Day 02 - Wi-Fi Security Checker

A simple Python-based Wi-Fi security auditing tool that analyzes the security posture of the connected wireless network.

---

## Features

- Detect Wi-Fi encryption type
- Display connected SSID
- Show signal strength
- Basic Wi-Fi security analysis
- Beginner-friendly implementation

---

## Technologies Used

- Python
- subprocess
- Wireless networking commands

---

## Installation

```bash
git clone <repo url>
cd Day-02-Wifi-Security-Checker
```

---

## Usage

```bash
python wifi_checker.py
```

---

## Example Output

```text
Connected Wi-Fi: HomeNetwork
Encryption: WPA2
Signal Strength: -48 dBm

Security Status:
[+] WPA2 detected
[+] Strong signal detected
```
