# Day 01 | Port Scanner

A simple Python based TCP port scanner that scans a target host for open TCP ports.

---

## Features

- Fast TCP port scanning
- Custom target support
- Detects open ports
- Beginner-friendly implementation
- Lightweight and easy to use

---

## Technologies Used

- Python
- Socket Programming

---

## Installation

```bash
git clone this repo url
cd Day-01-Port-Scanner
```

---

## Usage

```bash
python scanner.py
```

Enter:
- Target IP or domain
- Start port
- End port

Example:

```text
Target: scanme.nmap.org
Start Port: 1
End Port: 100
```

---

## Example Output

```text
Scanning Target: scanme.nmap.org
----------------------------------------
[+] Port 22 is OPEN
[+] Port 80 is OPEN
[+] Port 443 is OPEN

Scan Completed.
```
